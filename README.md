# Автоматизация исполнительной документации ПТО (Python Scripts)

Набор утилитарных скриптов на Python, разработанных для автоматизации рутинных процессов инженера ПТО в строительстве. Скрипты значительно ускоряют подготовку исполнительной документации и минимизируют человеческий фактор при заполнении форм.

> **Примечание по разработке:** Логика утилит спроектирована мной самостоятельно на основе реального опыта работы в ПТО. Скрипты созданы в парадигме **AI-assisted development** (с использованием ИИ для ускорения написания кода). Проекты адаптированы, протестированы и поддерживаются мной вручную.

## 🛠️ Функционал скриптов:

* **`aosr_generator_v2.py` / `aosr_generator_base.py`** — Скрипты для пакетной генерации Актов освидетельствования скрытых работ (АОСР). Автоматически подставляют данные, объемы, наименования работ и материалов из табличных баз данных в шаблоны документов.
* **`aosr_date_fixer.py`** — Утилита для автоматической проверки, корректировки и синхронизации дат в актах скрытых работ в соответствии со строительным журналом (исключает пересечения и логические ошибки в датах).
* **`registry_builder.py`** — Скрипт для автоматического формирования и структурирования общего Реестра исполнительной документации. Избавляет от ручного переноса номеров актов и документов.

## 🧰 Технологический стек:
* Python 3
* Модули для работы с офисными документами и таблицами (например, `python-docx`, `openpyxl` или `pandas` — *выберите/допишите свои библиотеки*)
* Модули файловой системы (`os`, `sys`, `shutil`)
