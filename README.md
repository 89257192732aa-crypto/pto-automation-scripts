# Автоматизация исполнительной документации ПТО и учебные скрипты (Python)

Набор скриптов на Python, включающий утилиты для автоматизации рутинных процессов инженера ПТО в строительстве (пакетная генерация АОСР, сборка реестров), а также базовые программы для отработки логических конструкций языка.

> **Примечание по разработке:** Логика утилит спроектирована мной самостоятельно на основе реального опыта работы. Скрипты созданы в парадигме **AI-assisted development** (с использованием ИИ для ускорения написания кода). Проекты адаптированы, протестированы и поддерживаются мной вручную.

## 🛠️ Функционал скриптов:

* **`aosr_generator_v2.py` / `aosr_generator_base.py`** — Скрипты для пакетной генерации Актов освидетельствования скрытых работ (АОСР). Автоматически подставляют данные, объемы и материалы из таблиц в шаблоны документов.
* **`aosr_date_fixer.py`** — Утилита для автоматической проверки, корректировки и синхронизации дат в актах скрытых работ в соответствии со строительным журналом.
* **`registry_builder.py`** — Скрипт для автоматического формирования и структурирования общего Реестра исполнительной документации.
* **`simple_calculator.py`** — Консольный калькулятор. Написан для практической отработки ветвлений `if-elif-else`, работы с пользовательским вводом `input()` и приведения типов данных.

## 🧰 Технологический стек:
* Python 3
* Модули файловой системы (`os`, `sys`, `shutil`)
* Работа с ветвлениями и базовыми типами данных Java/Python
